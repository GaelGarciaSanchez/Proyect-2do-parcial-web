Razor

razor no es un lenguaje de programacion per se, mas bien es un lenguaje de margenes del lado del servidor

razor es una sintaxis dedicada a margenes que te deja usar codigo embebido basado de el lado del servidor (como lo pueden ser visual basic o C#)a paginas Web;

el codigo basado en el servidor nos ayuda para crear contenido dinamico para web mientras se escribe junto a la pagina web que se vera despues en el navegador, cuando una pagina web es llamada el servidor ejecuta codigo basado en si dentro de la pagina antes de regresar la pagina de vuelta a el navegador para que el usuario pueda interactuar con ella, mientras que el servidor esta ejecutando este codigo mas eficientemente se pueden realizar tareas mas complejas como lo puede ser hacer conexiones a bases de datos
