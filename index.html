Desarrollo Web

podemos definir como desarrollo web al proceso de crear y mantener un sitio web que se mantiene activo en intenet meidante un hosting y puede ser desarrollado en diferentes lenguajes(javascript,css), cada sitio que se desarrolla tiene como identificador unico una URL que lo diferencia como dominio unico

ForntEnd: es el equipo de desarrollo que se dedica a la programacion que llega a la vista del cliente en este caso lo que seria la interfaz y los diseños de las paginas principales como esta, que estas viendo enfocandose principalmente en los diseños interactuables por los usuarios. |

Backend: este es el equipo que se dedica a la programacion o el desarrollo de todo aquello que esta relacionado con el servidor control de las bases de datos, accesos(en ciertos casos), actualizaciones y se encargan de parchar las actualizaciones mas importantes en los sitios |
